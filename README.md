# fable-mode

A collection of Claude skills that activate disciplined, high-rigor agentic behavior.

## Skills

- **fable-mode** (`SKILL.md`) — Fable-style agentic behavior: explicit multi-stage planning,
  sub-agent delegation, and self-verification. General-purpose, across any domain.
- **ultracode** (`ultracode/SKILL.md`) — Maximum-rigor coding mode. The fable-mode loop
  specialized for software: parallel codebase exploration via subagents, planned diffs,
  test-first implementation, and hard verification gates between stages.
