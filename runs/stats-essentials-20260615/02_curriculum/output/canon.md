# Canon — Statistics Essentials

Internal consistency contract for this mini. **Sits under** the series-canon
(`runs/statistics-20260615/series-canon.md`) — shared terms/metaphors/voice are defined there and are
binding; this file covers only this course's internal decisions. On a shared concept, the series-canon
wins. This course **establishes** the shared metaphors, so author them cleanly enough to be referenced.

## 1. Naming registry (course-local)
All shared terms come from the series-canon (population, sample, data, variation, spread, center,
distribution, probability, sampling variation, signal vs. noise, estimate, outlier). Course-local:
- **mean** = the balance-point average; **median** = the middle value. Pin "average" → **mean** on
  first use (M2). Never introduce "variance" or "standard deviation" as formulas — "standard
  deviation" appears once (M2) as a named *idea* ("a typical distance from the mean"), then "spread".

## 3. Metaphor-ownership map
Shared metaphors (defined in series-canon) are **established here**, one per concept — do not invent
new images for the same idea:
| Image | Concept | Established in |
|-------|---------|---------------|
| a crowd's heights | distribution / variation | M1–M2 |
| seesaw balance point | the mean (only) | M2 |
| wobble / scatter around a center | variation, spread, sampling variation | M1 |
| a handful of beans from a jar | sample vs. population | M3 |
| many coin flips over a long evening | probability / long-run frequency | M4 |
| signal through noise (radio static) | real pattern vs. random wobble | M4 |
No course-local metaphors beyond these; if a module needs a fresh concrete picture, keep it a one-off
and don't let it compete with a shared image.
