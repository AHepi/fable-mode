Continuity Without Limits

What you'll be able to do
  • Define the preimage of a set under a function.
  • State the topological definition of continuity (preimages of open sets are open).
  • Explain why this matches the intuitive "no sudden jumps / no tearing" picture.

You have probably been told what a continuous function is. Somewhere in a math class, someone drew a curve on a board and said: a function is continuous if you can draw its graph without lifting your pen. One unbroken stroke, no gaps, no jumps. It is a tidy picture, and for the graphs you met back then it even gave the right answers.

Forget it. Not because it is useless, but because it is a crutch we are about to throw away — and the throwing-away is the whole point of this chapter.

Here is the trouble. “Without lifting your pen” only makes sense for a curve you can actually draw: a function from the number line to the number line, with a graph you can trace. But topology lives in places far stranger than the plane — the surface of a doughnut, a Möbius band, spaces with no natural picture at all. There is no pen and no paper there. Worse, “drawing it in one stroke” quietly smuggles in the idea of a limit — the function “approaching” a value as you slide along — and limits are exactly the calculus machinery we promised to do without. So we need a notion of continuity that needs no graph, no pen, and no limits. We need to say what continuity is using only the one tool we have built so far: open sets.

It turns out we can. And the version we land on is sharper, stranger, and more powerful than the pen ever was.

From “no tearing” to “neighbors stay together”

Go back to the rubber-sheet image from module 01. A continuous reshaping of a shape — a stretch, a bend, a twist — is exactly one that does not tear. And what does “no tearing” really mean, point by point? It means the map keeps neighbors together. Two specks of clay sitting right next to each other before the stretch are still right next to each other after it. Nothing that was close gets flung far apart; that flinging-apart is precisely what a tear is.

So at heart, a continuous map is one that respects closeness: near things stay near. That is the honest content of the pen picture, stripped of the pen. Our job now is to say “stays near” in the language of open sets — because in module 02 an open set was exactly our way of talking about closeness. An open set around a point is the room around that point, the neighbors of where you’re standing.

Now picture a map fff that carries a space XXX over to a space YYY — every point of XXX gets sent somewhere in YYY. We want to test whether fff respects closeness. Here is the move that makes everything work, and it runs backward. Instead of standing in XXX and watching where points go, we stand in YYY, draw a patch of “room” — an open set — and ask: which points of XXX landed inside that patch? Then we check whether those points form a patch of room too.

That backward set — the points of XXX that land inside a chosen patch of YYY — is the one new idea this chapter rests on, so let us pin it down before we use it.

Definition (Preimage)

Let f:X→Yf : X \to Yf:X→Y be a function and let U⊆YU \subseteq YU⊆Y be any set. The preimage of UUU under fff, written f−1(U)f^{-1}(U)f−1(U), is the set of all points in XXX that fff sends into UUU:

f−1(U)={ x∈X:f(x)∈U }.f^{-1}(U) = \{\, x \in X : f(x) \in U \,\}.f−1(U)={x∈X:f(x)∈U}.

In words: f−1(U)f^{-1}(U)f−1(U) collects every point that lands in UUU — gather up all the starting points whose destination falls inside UUU, and that collection is the preimage.

A warning about the notation, because it is a genuine trap. That symbol f−1f^{-1}f−1 does not mean we have inverted fff or “undone” it. We are not running the function backward, and fff need not even have a way back. The symbol f−1(U)f^{-1}(U)f−1(U) is simply the name of a set — the set of points that land in UUU — and that set makes perfect sense for any function at all, no matter how badly behaved.

To feel why this is fine even when fff has no inverse, take the squaring map f(x)=x2f(x) = x^2f(x)=x2 on the number line. This function cannot be inverted as a whole: both 333 and −3-3−3 land on 999, so if someone hands you the output 999 there is no single input to hand back. And yet the preimage of, say, the open interval U=(4,9)U = (4, 9)U=(4,9) is no trouble at all. Which xxx satisfy 4<x2<94 < x^2 < 94<x2<9? Exactly those with 2<x<32 < x < 32<x<3 or −3<x<−2-3 < x < -2−3<x<−2. So

f−1((4,9))=(−3,−2)∪(2,3),f^{-1}\big((4,9)\big) = (-3,-2) \cup (2,3),f−1((4,9))=(−3,−2)∪(2,3),

a perfectly good set — in fact a union of two open intervals. We never inverted anything. We just asked “what lands in here?” and answered it. Keep this firmly in mind: in this chapter f−1f^{-1}f−1 names a preimage, never an inverse function. (Two chapters from now, when we meet maps that genuinely can be run backward, the same symbol will get its second job — and we will flag the switch loudly when it comes.)

Building the bridge

Now we can finish the thought. We wanted ”fff keeps neighbors together.” Translate it through the preimage.

Pick any patch of room in the destination — any open set U⊆YU \subseteq YU⊆Y. The points of XXX that land in that patch are f−1(U)f^{-1}(U)f−1(U). If fff truly respects closeness, then those starting points should themselves form a patch of room: wherever a point lands comfortably inside the open set UUU (not on its edge), the point it came from should sit comfortably inside an open set too, with neighbors all around it that also land in UUU. In other words, f−1(U)f^{-1}(U)f−1(U) should be open. If instead f−1(U)f^{-1}(U)f−1(U) had a sharp edge — a point with neighbors leaking outside it — that would be a place where fff tore the fabric, sending some near neighbors into UUU and others not. An open preimage is the fingerprint of no tearing.

So “keeps neighbors together,” said in the only language we have, becomes: the preimage of every open set is open. That is the definition.

Definition (Continuous map)

Let (X,τX)(X, \tau_X)(X,τX​) and (Y,τY)(Y, \tau_Y)(Y,τY​) be topological spaces. A function f:X→Yf : X \to Yf:X→Y is continuous if for every open set U⊆YU \subseteq YU⊆Y, the preimage f−1(U)f^{-1}(U)f−1(U) is open in XXX.

Read it slowly, because it is doing a lot in one line. It says nothing about pens, nothing about graphs, nothing about limits. It says only this: hand the map any open set in the target, and what comes back in the source is open too. Open sets in, open sets out — backward. That single condition, checked over all open sets, is what it now means for a map to be continuous. And notice it leans entirely on the open-set rules from module 03: “open” is whatever the topology declares to be open, so continuity is a relationship between two topologies, nothing more.

A tempting wrong answer: “Continuous should mean the image of every open set is open — open sets go forward to open sets.” It is the natural guess, and it is wrong. Continuity is about preimages, the backward direction, not forward images. The forward version describes a different and rarer property (an open map), and ordinary continuous functions routinely fail it. Take the flat map f(x)=5f(x) = 5f(x)=5 that sends every real number to 555. It is as continuous as a function can be — nothing moves, nothing tears. But its forward image of the open interval (0,1)(0,1)(0,1) is the single point {5}\{5\}{5}, which is not open. The definition has to run backward; that is not a detail to be tidied up later, it is the heart of the idea.

Worked example

The fastest way to trust the definition is to watch it pass a map it should pass and fail a map it should fail. We will use the simplest discontinuity there is — a jump — and a well-chosen open set to expose it.

Problem. Two maps from the real line to the real line.

  • The first is f(x)=2xf(x) = 2xf(x)=2x, a plain stretch that should be continuous.
  • The second is a step with a jump at the origin:
g(x)={0if x<0,1if x≥0.g(x) = \begin{cases} 0 & \text{if } x < 0, \\ 1 & \text{if } x \ge 0. \end{cases}g(x)={01​if x<0,if x≥0.​

Its graph sits flat at height 000 to the left, then leaps to height 111 at the origin — exactly the kind of tear we want continuity to reject. Use the preimage test to confirm fff is continuous on a sample open set and that ggg fails.

Solution. Recall that on the number line the open sets are built from open intervals (module 02), so we need only track what happens to open intervals.

The stretch f(x)=2xf(x) = 2xf(x)=2x. Pick any open interval in the target, say U=(2,6)U = (2, 6)U=(2,6). Which xxx land inside it? We need 2<2x<62 < 2x < 62<2x<6, that is 1<x<31 < x < 31<x<3. So

f−1((2,6))=(1,3),f^{-1}\big((2,6)\big) = (1, 3),f−1((2,6))=(1,3),

an open interval — open. And nothing special happened at (2,6)(2,6)(2,6): any open interval (a,b)(a,b)(a,b) pulls back to the open interval (a/2, b/2)(a/2,\, b/2)(a/2,b/2), and any open set (a union of such intervals) pulls back to a union of open intervals, which is open. Every open set has an open preimage, so fff is continuous. The stretch keeps neighbors together, just as the rubber-sheet picture promised.

The jump ggg. Here we hunt for a single open set whose preimage is not open — one bad set is all it takes to disqualify a map. The jump happens at the value 111, so aim an open patch right around it: let U=(0.5, 1.5)U = (0.5,\, 1.5)U=(0.5,1.5), an open interval that contains 111 but not 000. Which xxx does ggg send into UUU? Only the points where g(x)=1g(x) = 1g(x)=1, since g(x)=0g(x) = 0g(x)=0 misses UUU entirely. And g(x)=1g(x) = 1g(x)=1 exactly when x≥0x \ge 0x≥0. So

g−1((0.5,1.5))=[0,∞),g^{-1}\big((0.5, 1.5)\big) = [0, \infty),g−1((0.5,1.5))=[0,∞),

the half-line starting at and including 000. Is that open? No. The point 000 sits on the edge with no room to its left — every interval around 000 pokes out into the negative numbers, where ggg equals 000 and lands outside UUU. So 000 has neighbors that escape the patch: a tear, caught red-handed. The preimage [0,∞)[0,\infty)[0,∞) is not open, the definition is violated, and ggg is not continuous.

Look at what the definition just did. With no pen, no graph, no limit, it located the discontinuity precisely — at the one point x=0x = 0x=0 where the jump lives — purely by noticing that an open patch upstairs pulled back to a set with a sharp edge downstairs. The “no tearing” intuition and the preimage test agreed, exactly as we built them to.

Check yourself

Using the squaring map f(x)=x2f(x) = x^2f(x)=x2, what is the preimage f−1((−1,4))f^{-1}\big((-1, 4)\big)f−1((−1,4)) of the open interval (−1,4)(-1, 4)(−1,4)? (Careful — think about which real numbers xxx actually land inside that interval.)

We need every xxx with −1<x2<4-1 < x^2 < 4−1<x2<4. Since x2x^2x2 is never negative, the condition −1<x2-1 < x^2−1<x2 is automatic — it costs nothing. So the real constraint is x2<4x^2 < 4x2<4, which means −2<x<2-2 < x < 2−2<x<2. Therefore

Exercises

Work each one straight from the two definitions — preimage first, then the open-set test. Say why, not just what.

1. For the doubling map f(x)=2xf(x) = 2xf(x)=2x on the real line, compute the preimage f−1((−2,10))f^{-1}\big((-2, 10)\big)f−1((−2,10)).

Show solution

We need −2<2x<10-2 < 2x < 10−2<2x<10. Divide through by 222: −1<x<5-1 < x < 5−1<x<5. So f−1((−2,10))=(−1,5)f^{-1}\big((-2,10)\big) = (-1, 5)f−1((−2,10))=(−1,5). The preimage of an open interval under a stretch is again an open interval — which is one instance of the general fact that f(x)=2xf(x)=2xf(x)=2x is continuous: every open set pulls back to an open set.

2. Let h(x)=x+3h(x) = x + 3h(x)=x+3 (shift everything right by 3). Find h−1((0,1))h^{-1}\big((0, 1)\big)h−1((0,1)), and state in one sentence whether this is open.

Show solution

We need 0<x+3<10 < x + 3 < 10<x+3<1, so subtracting 333 gives −3<x<−2-3 < x < -2−3<x<−2. Thus h−1((0,1))=(−3,−2)h^{-1}\big((0,1)\big) = (-3, -2)h−1((0,1))=(−3,−2), which is an open interval, hence open. A shift never tears the line — it just slides it — so the preimage of any open set stays open.

3. Explain, in your own words, why we define continuity using the preimage (the backward direction) rather than “open sets get sent to open sets” (the forward direction). A one- or two-sentence reason, plus the constant map as an example, is enough.

Show solution

Continuity is about closeness being preserved: nearby inputs must have nearby outputs, with no tearing. The clean way to express that is to start from a patch of room in the target and require its incoming points to form a patch of room too — the preimage of an open set is open. The forward statement says something else entirely and even the simplest continuous maps fail it: the constant map f(x)=7f(x) = 7f(x)=7 never tears anything, yet it sends the whole open line to the single point {7}\{7\}{7}, which is not open. So “image of open is open” cannot be the definition of continuity; the backward direction is the right one.

5. (Conceptual.) The “draw it without lifting your pen” picture is the one many people carry from an earlier class. Give one reason that picture cannot serve as our definition of continuity in topology — and name what we replaced it with.

Show solution

The pen picture only works where there is a curve you can actually trace: a graph in the plane. Topology lives in places with no such picture — the surface of a doughnut, a Möbius band, a bare set of dots with no notion of distance — so “one unbroken stroke” has nothing to mean there. It also quietly relies on a function “approaching” a value as you slide along, which is the calculus idea of a limit, and we promised to build continuity without limits. We replaced the pen with the preimage test: fff is continuous when the preimage of every open set is open. That test needs only open sets, so it makes sense in any space at all.

6. (Conceptual.) Let f:X→Yf : X \to Yf:X→Y be a function and suppose the only open sets in YYY are ∅\varnothing∅ and YYY itself (the bare-minimum topology from module 03). Show that fff is automatically continuous, no matter what fff does.

Show solution

To check continuity we must show the preimage of every open set in YYY is open in XXX. But here YYY has only two open sets, so there are just two preimages to check. First, f−1(∅)f^{-1}(\varnothing)f−1(∅): which points of XXX land in the empty set? Second, f−1(Y)f^{-1}(Y)f−1(Y): which points land somewhere in YYY? Every point of XXX does, since fff sends each point somewhere in YYY. So f−1(Y)=Xf^{-1}(Y) = Xf−1(Y)=X, which is also open by that same axiom. Both preimages are open, so fff is continuous. The point: continuity is a relationship between two topologies, and if the target has almost no open sets, there is almost nothing to demand — every function clears the bar.

Recap

We threw out the pen. “Draw it without lifting your pen” was never the real idea of continuity; it was a picture that only works on graphs and that leans on limits we never wanted. What survives the throwing-away is plainer and tougher: a continuous map is one that keeps neighbors together, and the precise way to say so uses only open sets. Hand the map any open set in the target, look back at the points that land inside it — that backward set is the preimage f−1(U)f^{-1}(U)f−1(U), a name for a collection of points and never an inverted function — and demand that it be open. Preimage of every open set is open: that is continuity, no graph and no limit in sight. We watched a stretch pass the test and a jump fail it at the exact point where it tore. Next we put continuity to work on the question the whole course has been circling since the mug and the doughnut — what it really means for two shapes to be the same — where this same symbol f−1f^{-1}f−1 will quietly take on its second life as a genuine way back.
